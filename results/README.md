## Results

This directory contains the results corresponding to the experiments reported in the paper.

A more comprehensive and extended experimental evaluation of the same research line is available in the following repository:

https://github.com/iScOp-uniovi/Paper_NACO_Barredo_2023

Please refer specifically to the  
[`results/`](https://github.com/iScOp-uniovi/Paper_NACO_Barredo_2023/tree/main/results)  
section of that repository for the extended set of experimental results.

That repository is associated with the paper:

> *Precise makespan optimization via hybrid genetic algorithm for scientific workflow scheduling problem*  
> Natural Computing, 2023  
> Authors: Pablo Barredo, Jorge Puente  
> DOI: 10.1007/s11047-023-09950-5
